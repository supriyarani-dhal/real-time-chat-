## Run Locally

Clone the project

```bash
  git clone https://github.com/supriya626/real-time-chat-.git
```

Go to the project directory

```bash
  cd real-time-chat-
```

Install dependencies

```bash
  npm install
```
```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm start
```

Start the Client

```bash
  //open now terminal
  cd frontend
  npm start
```
